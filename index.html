NAC SOLUTION

Network Security

Virtual Solution

Network Access Control (NAC) is a term that encompasses various technologies developed to control/restrict access to the network by endpoint systems based on their "health status". The basic idea is that dangerous or vulnerable endpoint systems ("unhealthy") should not communicate on the enterprise network as they could introduce a security risk for critical processes and services. An NAC solution will prevent an unhealthy endpoint system from accessing the network normally until the health of that system is ensured.
